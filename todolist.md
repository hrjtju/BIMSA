# 总纲要

**目的**

对于如生命游戏及其变体这样的确定规则的演化系统，当规则域很大时，不可能暴力遍历所有规则组合以确定系统的真实动力学。神经网络的参与可以帮我们简化这个过程：通过学习一步转移映射，神经网络可以有效学习生命游戏中决定元胞下一时刻状态的邻域大小。此外，神经网络还能利用学习到的一步映射，通过特定的输入直接得到神经网络所预测的对应规则。使用神经网络预测得到完整的规则集合后，可以通过于训练数据对比验证的方式保证规则集合于训练数据相容。

**流程**

1. 获取用于训练的数据 / 给定规则，生成用于训练的数据。
2. 使用神经网络在数据上进行训练。
    1. 输入为系统总状态（矩阵）$x_t$，输出为估计的下一状态 $x_{t+1}$。
    2. 输入的形状为 `[batch, 1, w, h]`，输出形状为 `[batch, 2, w, h]`，其中第二维度为像素在下一时刻属于存活或是死亡状态的似然值 `[live, dead]`。
    3. 损失函数由逐像素点的交叉熵损失和作用于权重的 L1 正则化损失组成
    4. 对称性损失通过在数据标签对中人为添加做相同的环面意义上的随机旋转和平移变换的数据组成。如选定的平移变换和旋转变换是 $T$ 和 $R$，则组合后的输入和标签分别为 $[x, T\circ R(x)]$ 和 $[y, T\circ R(y)]$

3. 获取邻域形状。
    1. 绘制网络首层卷积核权重可视化图像
    2. 绘制网络平均最大有效感受野绝对值（如果邻域实在不对称难一遍别，则对其按 Chebyshev 距离环做平均）
    3. 比对 (3.1) 和 (3.2)，划定阈值后输出邻域形状
    4. 确定邻域密度函数 $A_N$，对于只有一层的邻域，$A_N$ 是存活细胞数目。但对于多层邻域的情况尚未讨论
4. 预测规则。
    a.通过邻域统计预测规则。
        1. 统计转移三元组 $(A_N(x_t), x_t, x_{t+1})$ 并做统计。
        2. 根据固定三元组的前两个元素做分类，每一类分别统计下一状态被预测为存活或是死亡的频数
        3. 根据频数比例的阈值和似然的方法确定规则
    b. 通过神经网络生成规则
        1. 在数据中采样符合邻域大小的矩形窗口，并按照邻域密度函数 $A_N$ 做调整，使得采样出来是均匀分布
        2. 对采样得到的窗口做环面意义上的随机平移和旋转
        3. 输入神经网络，同一个 $A_N$ 的输出似然取平均值，然后二值化得到生成的规则
5. 根据实际数据修正规则。
    1. 选定几条训练数据中的演化轨道（要求演化步数足够大）
    2. 利用 (4) 中得到的规则从每条演化轨道的初始状态启动演化
    3. 对比模拟轨道和真实轨道，找到所有这样的邻域 $N$: $t$ 时刻模拟和真实轨道的邻域相同，而 $t+1$ 时刻中心细胞的预测结果不同。
    4. 根据 (5.3) 中的不同直接修改规则，使得规则和数据相容。
    5. 不断重复 (5.4) 和 (5.3)，使得修改后的结果在所选轨道上的所有演化相容。
